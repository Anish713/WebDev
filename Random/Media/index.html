HTML Entities

HTML entities are used to display reserved characters in HTML.

Reserved characters in HTML must be replaced with character entities.

Characters that are not present on your keyboard can also be replaced by entities.

Some characters are reserved in HTML.

If you use the less than (<) or greater than (>) signs in your text, the browser might mix them with tags.

Character entities are used to display reserved characters in HTML.

A character entity looks like this:

&entity_name;

or

&entity_number;

When Should You Use HTML Entities?

Some reserved characters have to be replaced with character entities when you use them in HTML documents.

The entities are case sensitive!

Always use the entity name when writing HTML code for a webpage.

Tip: The entity name is case sensitive.

Tip: The entity number is case sensitive!

Eg: paragraph tag in html is given as: <p>Lorem ipsum dolor sit amet.</p>
